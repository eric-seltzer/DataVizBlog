Intoduction

For this blog post, I wanted to take on more of a challenging idea which was to both create a prediction model and visualize that data. To to this, I also wanted to combine this with my passion for sports, particularly basketball. It felt only right to use NBA data to accomplish this. On Data World, I found a data set that had team stats from the 97-98 season all the way until the 21-22 season.

Building a model

I decided to use some of the metrics in the data set to predict whether a team would make the playoffs or not. I made a response variables Playoffs by using a condition of if a team played more than 82 games or not in season. I also made another qualitative variable in over100 which shows whether a team averaged over 100 points or not for that season. This makes the visualization aspect more fun to do.

In order to make sure that this model isn’t testing using data that knows the answer, we should get rid of both of the No and G variables. We also don’t care about what team it is so we should get rid of the team factor.

team_stats_tidy <-
  team_stats_tidy |>
  select(-(c("No","G", "Team")))

As this is a blog for data visualization, I am only going to choose four predictors that were all found to be relatively significant in a model using all the predictors. Two of these are qualitative to demonstrate more visualization. The predictors are Deff, Conference, FG_pct, and over100. Shown below is my model with their coefficients and p values.

playoffs_mod <- glm(Playoffs ~ Deff + Conference + FG_pct + over100,
                    data = team_stats_tidy,
                    family = 'binomial')

For my visualization I plan on using the geom_rug feature, so I need to make two separate data frames with teams that made the playoffs and teams that didn’t.

stats_playoffs <-
  team_stats_tidy |>
  filter(Playoffs == 1)

stats_nplayoffs <-
  team_stats_tidy |>
  filter(Playoffs == 0)

Conclusion

This plot shows the differences in the probability of a team to make the playoffs dependent on their efficiency difference, their field goal percentage, their conference, and if they averaged over 100 points or not. What we can see here is that for a team in the Eastern conference, they need a lower efficiency difference to have an over .5 probability of making the playoffs. As well, a team’s probability of making the playoffs is higher if they averaged over 100 points.
